# 📖 Asset Manager 사용 가이드

> 정보자산 통합 관리 시스템 — 사용자 매뉴얼
> 최종 업데이트: 2026-03-10

---

## 🔐 로그인

1. 브라우저에서 `http://<서버주소>:8080` 접속
2. **사용자명**과 **비밀번호** 입력 후 로그인
3. 최초 로그인 시 비밀번호를 반드시 변경하세요

> ⚠️ 비밀번호 5회 이상 실패 시 계정이 잠깁니다. 관리자에게 문의하세요.

---

## 📋 라이선스 관리

### 라이선스 목록 조회
- 메뉴: **라이선스** 클릭
- 검색/필터로 원하는 라이선스를 빠르게 찾을 수 있습니다

### 라이선스 등록
1. **라이선스 > 등록** 클릭
2. 필수 항목 입력:
   - **라이선스명** (예: Microsoft 365, Figma)
   - **유형** (구독형 / 영구형 / 시트형 등)
   - **만료일** (구독 갱신 알림 기준)
   - **총 수량** (시트 기반의 경우)
3. **등록** 버튼 클릭

### 라이선스 할당
1. 라이선스 상세 페이지 진입
2. **할당** 버튼 클릭
3. 조직원 선택 후 확인

### 라이선스 반납
1. 라이선스 상세 페이지 진입
2. 할당된 조직원 목록에서 **반납** 클릭

---

## 👥 조직원 관리

### 조직원 조회
- 메뉴: **조직원** 클릭
- 이름/부서/이메일로 검색 가능

### 조직원 등록
1. **조직원 > 등록** 클릭
2. 필수 항목 입력:
   - 이름, 이메일, 부서, 직책
   - 소속 조직 단위 선택
3. **등록** 버튼 클릭

> ✅ 등록 시 해당 그룹의 기본 라이선스가 자동 할당됩니다

### 조직원 퇴사 처리
1. 조직원 상세 페이지 진입
2. **퇴사 처리** 클릭
3. 확인 — 퇴사 후 **7일간 유예** 후 자동 삭제됩니다

## 📂 CSV 가져오기

여러 명의 조직원 또는 라이선스를 한 번에 등록할 때 사용합니다.

1. 메뉴: **설정 > CSV 가져오기** 클릭
2. **템플릿 다운로드** 후 양식에 맞게 작성
3. 파일 업로드 후 **가져오기** 클릭

### CSV 양식 작성 주의사항
| 항목 | 주의사항 |
|---|---|
| 날짜 | `YYYY-MM-DD` 형식 (예: 2026-03-10) |
| 조직 단위 | 시스템에 등록된 이름과 정확히 일치해야 함 |
| 그룹명 | 없으면 기본 그룹에 자동 배정 |
| 이메일 | 중복 불가, 형식 준수 (xxx@xxx.com) |

## 👑 관리자 전용 기능 (ADMIN)

### 사용자 관리
- 메뉴: **관리 > 사용자 관리** 클릭
- 새 사용자 등록 / 역할 변경 / 임시 비밀번호 발급 가능

### 역할 종류
| 역할 | 권한 |
|---|---|
| **ADMIN** | 모든 기능 + 사용자 관리 |
| **USER** | 조회 및 기본 업무 처리 |

## ❓ 자주 묻는 질문 (FAQ)

### Q. 비밀번호를 잊어버렸어요
→ 관리자에게 문의하면 **임시 비밀번호**를 발급받을 수 있습니다.

### Q. 라이선스 만료 알림이 오지 않아요
→ 라이선스에 **만료일**이 등록되어 있는지 확인하세요.
→ 담당자가 지정되어 있어야 알림이 발송됩니다.

### Q. CSV 가져오기가 실패해요
→ 날짜 형식이 `YYYY-MM-DD`인지 확인하세요.
→ 조직 단위명이 시스템에 등록된 이름과 정확히 일치하는지 확인하세요.
→ 이메일 중복 여부를 확인하세요.

### Q. 조직원 퇴사 처리 후 바로 삭제가 안 돼요
→ 퇴사 후 **7일간 유예 기간**이 있습니다. 7일 후 자동 삭제됩니다.
→ 즉시 삭제가 필요하면 관리자에게 문의하세요.

### Q. 라이선스 할당이 안 돼요 (시트 부족)
→ 라이선스의 **총 수량**을 확인하고, 필요 시 수량을 늘려야 합니다.

### Q. 페이지가 느려요
→ 브라우저 캐시를 초기화하거나 (`Ctrl+Shift+R`) 다른 브라우저로 시도해 보세요.
